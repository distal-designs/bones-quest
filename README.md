# Bones Quest

## Setup

### Ubuntu/Debian

Run `sudo apt install libasound2-dev libsdl2-dev build-essential pkg-config` to install necessary
dependencies.

### Solus

Run `sudo eopkg install -c system.devel sdl2-devel alsa-lib-devel` to install necessary
dependencies.
